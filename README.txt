Linkdeki keras modeli indirilip "Exercise Predict" klasörünün içine atılmalıdır.

1-Sanal ortamı oluşturma
    python -m venv myenv


2-Sanal ortamı aktifleştirme
    Windows için:
        myenv\Scripts\activate


    macOS ve Linux için:
        source myenv/bin/activate


3-Gerekli kütüphaneleri yükleme
    pip install -r requirements.txt


4-Projeyi çalıştırma
    .\myenv\Scripts\python.exe main.py


(Her bir egzersizden sonra çıkmak için q tuşuna basılmalıdır.)
